¿Como usar el compilador?

  1. Elige entre subir un .zip o seleccionar una carpeta completa.
  2. El sistema detectará automáticamente el nombre del recurso y todos los archivos .lua (cliente, server, config, etc.).
  3. Marca los archivos que deseas compilar (por defecto todos están seleccionados).
  4. Haz clic en COMPILAR Y DESCARGAR.
  5. El sistema compila usando luac.mtasa.com y actualiza el meta.xml.
  6. Sube el ZIP resultante a tu servidor MTA ¡Listo!
